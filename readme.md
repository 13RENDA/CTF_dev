# CTF Development Repository

CTF challenge development workspace.

## Structure

- `christmas-vibe-check/` - Crypto challenge (Python/Flask)
- `cmgr/` - Challenge manager artifacts
- `cmgr.db` - Challenge database

## Usage

Each challenge directory contains:
- `main.py` - Challenge application
- `problem.md` - Challenge description
- `setup_challenge.py` - Setup script
- `solve.py` - Reference solution
- `Dockerfile` - Container configuration
